Инструкция по установке:
  1. Скачайте проект по кнопке "Скачать".
  2. Распакуйте архив WantToKnowEverythink.rar.
  3. Откройте папку "Тестовый комплекс".
  4. Запустите CyberBrain.exe.
  5. Для дальнейшей работы необходимо войти в систему под одним из тестовых аккаунтов, которые прописаны на форме входа.
  6. Программа предлагает три варианта проверки знаний (тесты, карточки, соотношение).
  7. После выбора варианта проверки выбирайте категорию знаний и проходите уровни.
  8. Приятного тестирования!!!
